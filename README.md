# U0U Landing Page

Simple landing page for U0U beta waitlist.

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect repo to Netlify
3. Deploy

Netlify handles form submissions automatically (no backend needed).

## Files

- `index.html` - Landing page
- `success.html` - Post-signup confirmation
- `styles.css` - Styling
- `app.js` - Optional form UX enhancements
- `netlify.toml` - Netlify configuration

## Form Submissions

View submissions in Netlify dashboard under **Forms > waitlist**.
